Eine schlanke Toolbar für WordPress mit Zoom- und High-Contrast-Funktion. Einstellungen werden im Browser gespeichert.

== Description ==
**Accessibility Toolbar Lite** ergänzt eine schmale Toolbar oben (oder unten) in deinem Theme. Besucher:innen können:
- Zoom In/Out in 10-%-Schritten  
- High-Contrast-Modus aktivieren/deaktivieren  
Einstellungen (Zoom, Kontrast, Position) werden im LocalStorage gespeichert.

== Installation ==
1. Plugin-Ordner nach `/wp-content/plugins/accessibility-toolbar-lite/` hochladen.  
2. Unter **Plugins** aktivieren.  
3. Unter **Einstellungen → Accessibility Toolbar** Default-Werte festlegen.  
4. Live-Frontend aufrufen – die Toolbar erscheint.

== Frequently Asked Questions ==
= Ist das Plugin GDPR-konform? =  
Ja, es verwendet kein Tracking und keine externen Services.

= Funktioniert es mit jedem Theme? =  
Ja, sofern dein Theme `wp_body_open()` oder `wp_footer()` aufruft.

= Wie speichert das Plugin Einstellungen? =  
Im LocalStorage des Browsers.
